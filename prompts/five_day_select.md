# 五天科技新闻筛选 Prompt

读取 `data/daily/` 中最近 5 天的科技新闻。如果本地归档不足，使用最新网络信息补齐最近 5 天的重要科技新闻。

目标：
- 面向“普通人看懂的一周科技大事”抖音账号。
- 对新闻进行去重、聚类、核验和评分。
- 筛出 8-12 条候选新闻。
- 推荐 5-7 条进入本期视频。

每条候选新闻包含：
- 推荐排序
- 标题
- 事件摘要
- 为什么重要
- 对普通人的影响
- 适合视频讲述的角度
- 来源链接
- 入选分数
- 风险提醒

产物：
- 结构化结果保存到 `data/selected/YYYY-MM-DD-selection.json`。
- 可读 Markdown 报告保存到 `reports/YYYY-MM-DD-five-day-selection.md`。
